Migration Windows 10 vers Windows 11

Gestion de parc – Stage 1ère année – Grand Cognac

📌

Présentation

Dans le cadre de mon stage au service informatique de Grand Cognac, j'ai participé à la migration des postes de travail de Windows 10 vers Windows 11, motivée par la fin de support de Windows 10 prévue par Microsoft.

Procédure : sélection des postes éligibles depuis un fichier Excel → vérification dans GLPI → ajout au groupe GRP_POLICY_TESTW11UPDATE_DEVICES dans Azure Active Directory → déploiement automatique via politique Microsoft Intune → synchronisation du PC.
